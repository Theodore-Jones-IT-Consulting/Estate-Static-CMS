Shortcode Script Usage

This section demonstrates the use of a shortcode script. The shortcode [script:greeting name=Jane] calls the 'greeting.py' script located in the 'scripts' subfolder. It passes the parameter 'name' with the value 'Jane' to the script. The script returns a greeting message, which is then dynamically inserted here.

Hello, Jane!
